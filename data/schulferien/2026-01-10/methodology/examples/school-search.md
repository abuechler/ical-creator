# Example: Gemeinde → Schule → Ferien Pattern

## When to Use

This is the most common pattern for finding Ferienpläne. Use when the Gemeinde website doesn't have the Ferienplan directly.

## Example: Höri

### Search Path

1. **Start:** https://www.hoeri.ch (Gemeinde website)
2. **Navigate:** Bildung section
3. **Find:** Link to school website (www.schulehoeri.ch)
4. **On school site:** Informationen → Ferienplan
5. **Result:** PDF download available

### Step-by-Step

```
www.hoeri.ch
    └── Bildung
        └── "Weitergehende Informationen finden Sie unter www.schulehoeri.ch"
            └── www.schulehoeri.ch
                └── Informationen
                    └── Ferienplan
                        └── PDF download ✓
```

## Example: Hochfelden

### Search Path

1. **Start:** https://www.hochfelden.ch
2. **Search:** "Schule" on the page
3. **Find:** Link to https://www.schule-hochfelden.ch
4. **Navigate:** Downloads section
5. **Find:** Ferienplan in document list

### Key Insight

The Downloads section wasn't labeled "Ferienplan" - it was under general "Downloads" with the Ferienplan listed among other documents.

## Tips for This Pattern

1. Look for "Bildung" or "Schulen" in Gemeinde navigation
2. School website URL is often linked in text, not just menu
3. On school site, try multiple navigation paths:
   - Organisation → Ferienplan
   - Informationen → Ferienplan
   - Downloads / Dokumente
4. Use browser search (Ctrl+F) for "Ferien"
