# Event Mood / Energy Tag

## Description
Tag events with expected mood/energy level using emoji sliders, helping plan balanced days.

## User Value
- **Balance**: Avoid scheduling all draining meetings together
- **Self-awareness**: Track what energizes vs drains you
- **Planning**: Space out high-energy events
