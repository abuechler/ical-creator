# Event Statistics Dashboard

## Description
Show statistics about saved events - total count, events by type, most used locations, etc.

## User Value
- **Insights**: See patterns in your events
- **Overview**: Quick summary of all events
- **Interesting**: Discover usage patterns

## Implementation Details
- Add collapsible "Statistics" section
- Show: Total events, recurring vs one-time
- Most frequent locations
- Events by frequency type (daily/weekly/monthly)
- Average event duration
- Events created this month
