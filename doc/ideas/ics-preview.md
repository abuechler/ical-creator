# ICS File Preview

## Description
Show a preview of the raw .ics file content before downloading, useful for debugging and learning the format.

## User Value
- **Learning**: See how iCal format works
- **Debugging**: Verify correct output
- **Advanced users**: Check for issues before import

## Implementation Details
- Add "Preview .ics" button next to Download
- Show modal with syntax-highlighted .ics content
- Copy to clipboard button in preview
- Collapsible in debug section for power users
