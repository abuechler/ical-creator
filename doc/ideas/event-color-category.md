# Event Color/Category

## Description
Add color picker or category selector to events for visual organization in the calendar preview.

## User Value
- **Organization**: Visual distinction between event types
- **Recognition**: Quickly identify work vs personal events
- **Aesthetics**: More colorful, engaging calendar

## Implementation Details
- Add color picker or category dropdown in form
- Categories: Work (blue), Personal (green), Health (red), Travel (orange), Other (gray)
- Color appears as dot/bar in calendar preview
- Color stored in event data and .ics file (X-APPLE-CALENDAR-COLOR)
