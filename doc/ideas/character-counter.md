# Character Counter for Description

## Description
Show character/word count for description field with visual indicator when approaching limits.

## User Value
- **Awareness**: Know how long description is
- **Limits**: Some calendars truncate long text
- **Writing**: Help keep descriptions concise

## Implementation Details
- Show "X characters" below description field
- Optional word count toggle
- Warning color when > 500 chars
- Recommended limit indicator
- Update count as user types
