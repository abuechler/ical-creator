# Random Event / Activity Suggester

## Description
"I'm bored" button that generates random activity suggestions based on time of day, weather, and preferences.

## User Value
- **Spontaneity**: Break routine with new ideas
- **Fun**: Playful feature for personal use
- **Discovery**: Try new activities

## Implementation Details
- "Surprise Me" button in header
- Categories: Indoor, Outdoor, Social, Solo, Active, Relaxing
- Suggests activity based on:
  - Time of day (morning jog, evening movie)
  - Weather (if integrated)
  - User preferences (saved categories)
- Creates event with suggested activity
- "Another one" to regenerate
- User can save favorite suggestions
