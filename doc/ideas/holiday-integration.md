# Holiday Indicator in Calendar

## Description
Show public holidays in the calendar preview to help users avoid scheduling on holidays.

## User Value
- **Awareness**: See holidays when picking dates
- **Planning**: Avoid scheduling on public holidays
- **Context**: Better calendar overview

## Implementation Details
- Add small indicator for common holidays
- Support major holidays (Christmas, New Year, Easter)
- Region-based holiday selection (US, EU, etc.)
- Holidays shown as subtle markers in calendar grid
- Use static holiday data (no API needed)
