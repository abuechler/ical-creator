# Timezone Converter Display

## Description
Show the event time converted to multiple timezones simultaneously, helpful for scheduling across time zones.

## User Value
- **Global teams**: See time in colleague's timezone
- **Travel**: Check event time at destination
- **Clarity**: Avoid timezone confusion

## Implementation Details
- Add "Show in other timezones" expandable section
- Display event time in 2-3 other timezones
- Let user pick which timezones to show
- Quick add: "Add my timezone" based on browser
- Persist selected timezones in localStorage
