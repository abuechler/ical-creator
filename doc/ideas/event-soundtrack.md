# Event Soundtrack / Ambiance

## Description
Suggest or embed ambient sounds/music playlists for events - focus music for work blocks, party playlists for celebrations.

## User Value
- **Atmosphere**: Set the mood for events
- **Productivity**: Focus music for work blocks
- **Fun**: Party playlists for celebrations

## Implementation Details
- Dropdown: Focus, Energetic, Calm, Celebration, None
- Links to Spotify/YouTube playlists
- Add as URL in event description
- Suggestions based on event type/keywords
- Preview audio clips in app
- Optional: ambient sounds player built-in
