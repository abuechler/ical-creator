# Confetti Celebration

## Description
Show a subtle confetti animation when successfully downloading an event, adding delight to the experience.

## User Value
- **Delight**: Fun, memorable interaction
- **Feedback**: Clear success indication
- **Personality**: Makes app feel friendly

## Implementation Details
- Trigger on successful .ics download
- Subtle, short animation (1-2 seconds)
- Confetti in brand colors (teal, gray)
- Option to disable in settings
- Respect prefers-reduced-motion
- Use canvas or CSS animation
