# Habit Streak Tracking

## Description
For recurring events, track and display completion streaks. Mark events as "done" and see your streak counter.

## User Value
- **Motivation**: Visual streak encourages consistency
- **Gamification**: Satisfying to maintain streaks
- **Habits**: Perfect for gym, meditation, learning

## Implementation Details
- "Track as habit" toggle for recurring events
- Saved events show streak counter
- Mark individual occurrences as complete
- Streak breaks shown (missed dates)
- Calendar shows completed days differently
- Stats: current streak, longest streak, completion %
- Optional streak milestone celebrations
