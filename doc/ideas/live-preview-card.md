# Live Event Card Preview

## Description
Show a live preview card of how the event will appear in calendar apps (Google Calendar, Apple Calendar style).

## User Value
- **WYSIWYG**: See exact appearance before download
- **Confidence**: Know what you're creating
- **Visual feedback**: Changes update instantly

## Implementation Details
- Card styled like Google Calendar event popup
- Updates in real-time as form changes
- Shows: Title, time, location, description preview
- Recurrence badge if applicable
- Reminder indicator
- Position next to or below form
