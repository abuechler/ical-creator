# Progress Stepper/Wizard

## Description
Replace single-page form with a step-by-step wizard showing progress through event creation stages.

## User Value
- **Guided**: Clear path through complex forms
- **Less overwhelming**: One section at a time
- **Progress**: See how far along you are

## Implementation Details
- Steps: 1. Basic Info → 2. Date/Time → 3. Recurrence → 4. Review
- Progress bar at top showing current step
- Next/Back buttons to navigate
- Step indicators show completion status
- Allow jumping to completed steps
- Final step shows summary before download
