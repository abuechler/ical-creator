# Keyboard Shortcuts

## Description
Add keyboard shortcuts for common actions to improve power user efficiency.

## User Value
- **Speed**: Faster workflow for frequent users
- **Accessibility**: Alternative input method
- **Pro users**: Expected feature for productivity apps

## Implementation Details
- Shortcuts:
  - `Ctrl/Cmd + S` - Download .ics file
  - `Ctrl/Cmd + N` - New event (clear form)
  - `Escape` - Close modals
  - `Ctrl/Cmd + D` - Toggle dark mode (if implemented)
- Show shortcuts hint in footer or help modal
- Don't interfere with browser shortcuts
