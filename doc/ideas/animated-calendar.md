# Animated Calendar Transitions

## Description
Add smooth animations when navigating between months or when event dates change in the calendar preview.

## User Value
- **Polish**: Professional, refined feel
- **Clarity**: Visual feedback on changes
- **Engagement**: More enjoyable to use

## Implementation Details
- Slide animation when changing months
- Pulse/highlight animation on event date changes
- Fade transition when toggling recurring events
- CSS transitions for smooth rendering
- Respect prefers-reduced-motion setting
