# Recent Locations Autocomplete

## Description
Remember previously entered locations and offer them as autocomplete suggestions when typing in the location field.

## User Value
- **Speed**: Quickly select previously used locations
- **Accuracy**: Consistent location names
- **Convenience**: No retyping common venues

## Implementation Details
- Store unique locations in localStorage (max 10-20)
- Show dropdown with matching locations as user types
- Click to select or continue typing custom
- Clear recent locations option in debug section
