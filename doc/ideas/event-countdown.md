# Event Countdown Display

## Description
Show a live countdown timer to the event start time in the preview section.

## User Value
- **Excitement**: Build anticipation for events
- **Context**: See how soon event is
- **Useful**: Know time remaining at a glance

## Implementation Details
- Show "Starts in X days, Y hours" in preview
- Update every minute (or second if < 1 hour)
- Different displays: "in 3 days", "in 5 hours", "in 20 minutes"
- Show "Started" or "Ended" for past events
- Color coding: green (soon), gray (far away)
