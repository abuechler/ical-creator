# Fullscreen/Focus Mode

## Description
Add a distraction-free fullscreen mode that hides saved events and debug info, showing only the form.

## User Value
- **Focus**: Concentrate on current event
- **Clean**: Minimal distraction UI
- **Presentation**: Clean view for demos

## Implementation Details
- Toggle button in header (expand icon)
- Hides: Saved Events, Debug Info, Footer
- Centers form with max-width
- Press Escape to exit
- Keyboard shortcut: F11 or Cmd+Shift+F
