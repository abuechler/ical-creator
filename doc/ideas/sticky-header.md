# Sticky Header with Event Title

## Description
Make the header sticky and show the event title being edited as you scroll down the form.

## User Value
- **Context**: Always know what event you're editing
- **Navigation**: Header stays accessible
- **Branding**: App identity always visible

## Implementation Details
- Header becomes sticky on scroll
- Compact version when scrolled (smaller padding)
- Show event title in header when editing
- Smooth transition to compact mode
- Shadow appears when sticky
