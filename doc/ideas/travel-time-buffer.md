# Travel Time Buffer

## Description
Option to add travel time before/after events, creating buffer time blocks in the calendar.

## User Value
- **Realistic**: Account for commute time
- **Planning**: Avoid back-to-back meetings
- **Awareness**: Remember travel is needed

## Implementation Details
- "Add travel time" checkbox
- Duration options: 15, 30, 45, 60 minutes
- Creates separate buffer event OR adjusts reminder
- Buffer shown as "Travel to [Event]"
- Optional return travel time after event
- Could integrate with location for smart suggestions
