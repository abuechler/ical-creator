# World Clock Visualizer

## Description
Interactive globe or world map that shows event time across multiple timezones visually, with day/night indication.

## User Value
- **Global teams**: Schedule across timezones intuitively
- **Visual**: See who's in daytime vs nighttime
- **Memorable**: Unique, impressive feature

## Implementation Details
- Mini world map or globe visualization
- Mark selected timezones as pins
- Show event time at each pin
- Day/night terminator line
- Click timezone to convert
- Highlight "working hours" overlay
- SVG-based, no heavy 3D libraries
- Collapsible for those who don't need it
