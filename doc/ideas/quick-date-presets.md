# Quick Date Presets

## Description
Add quick-select buttons for common dates like "Today", "Tomorrow", "Next Week" to speed up date entry.

## User Value
- **Speed**: One-click date selection
- **Convenience**: No need to navigate date picker
- **Common use case**: Most events are near-future

## Implementation Details
- Add preset buttons below date fields: Today | Tomorrow | Next Week | Next Month
- Clicking preset fills both start date and adjusts end date
- For recurring events, also set sensible end date
- Style as small pill buttons
