# Event Chaining / Sequences

## Description
Create linked event sequences where completing one event automatically sets up the next (e.g., "Workout → Shower → Breakfast").

## User Value
- **Routines**: Build morning/evening routines
- **Workflows**: Sequential task planning
- **Automation**: Events flow naturally

## Implementation Details
- "Add next event" button after creating event
- Chain multiple events in sequence
- Auto-calculate start times based on durations
- Download as single .ics with multiple VEVENTs
- Visual chain preview showing flow
- Name the chain: "Morning Routine"
- Optional gap time between events
