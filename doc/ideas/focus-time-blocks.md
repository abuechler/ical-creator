# Focus Time / Deep Work Blocks

## Description
Quick button to create "Focus Time" blocks - distraction-free work periods with auto-decline meeting suggestions (via calendar busy status).

## User Value
- **Productivity**: Protect time for deep work
- **Work-life balance**: Block personal time
- **Calendar hygiene**: Show as "busy" to others

## Implementation Details
- "Create Focus Block" quick action
- Pre-set durations: 1h, 2h, 4h Focus Time
- Sets event as BUSY with X-MICROSOFT-CDO-BUSYSTATUS
- Default title: "Focus Time - Do Not Disturb"
- Optional: different colors for focus vs meetings
- Integrates with daily/weekly recurring
