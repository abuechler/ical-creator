# Event Image/Banner

## Description
Allow users to attach an image or select a banner that gets embedded or linked in the event.

## User Value
- **Visual**: Events with images are more memorable
- **Context**: Add maps, flyers, or reference images
- **Professional**: Great for invitations

## Implementation Details
- Image upload or URL input
- Stored as base64 in description (small images)
- Or store as URL reference in description
- Preview thumbnail in form
- Drag-and-drop upload support
- Resize/compress large images
- Add to ATTACH property in .ics
