# Batch Download Events

## Description
Select multiple saved events and download them all as a single .ics file or as a zip of individual files.

## User Value
- **Efficiency**: Export multiple events at once
- **Backup**: Quick backup of selected events
- **Organization**: Group related events together

## Implementation Details
- Add checkboxes to saved event cards
- "Download Selected" button appears when events selected
- Option: Single .ics with multiple VEVENTs or ZIP file
- Select all / Deselect all buttons
- Download count indicator
