# Event Conflict Warning

## Description
Warn users when a new event overlaps with existing saved events at the same time.

## User Value
- **Awareness**: Know about scheduling conflicts
- **Prevention**: Avoid double-booking
- **Planning**: Better time management

## Implementation Details
- Check new event time against saved events
- Show warning if overlap detected
- Warning shows conflicting event name/time
- Allow user to proceed anyway
- Check both one-time and recurring events
- Consider timezone when comparing
