# Time Blocking Mode

## Description
Switch to a visual time-blocking interface where you drag to create events directly on a day/week timeline view.

## User Value
- **Visual planning**: See your day at a glance
- **Intuitive**: Drag to create, resize events
- **Overview**: Better than form-based entry for planning days

## Implementation Details
- Toggle between Form Mode and Timeline Mode
- Day view: 24h vertical timeline
- Drag to create new event block
- Resize blocks to adjust duration
- Click block to edit details
- Color-coded blocks
- Export all blocks as single .ics
- Great for daily planning sessions
