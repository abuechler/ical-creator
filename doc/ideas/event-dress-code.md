# Dress Code / What to Bring

## Description
Add a "What to bring" or "Dress code" field that appears as a checklist before the event.

## User Value
- **Preparation**: Remember what to bring
- **Clarity**: Communicate expectations to attendees
- **Organization**: Pre-event checklist

## Implementation Details
- New field: "What to bring / Dress code"
- Preset suggestions: Casual, Business, Formal, Costume
- Custom checklist items
- Added to event description in formatted way
- Shows as bullet list in calendar apps
- Great for: gym (towel, water), meetings (laptop), parties (gift)
