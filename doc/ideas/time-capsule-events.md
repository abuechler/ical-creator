# Time Capsule Events

## Description
Create "future self" events - messages or goals you set for a future date, revealed when the time comes.

## User Value
- **Reflection**: Messages to future self
- **Goals**: Set and review yearly goals
- **Surprise**: Delightful rediscovery

## Implementation Details
- "Create Time Capsule" option
- Set reveal date (1 month, 1 year, custom)
- Write message/goals in description
- Event appears on reveal date
- Description hidden until then (just shows "Time Capsule")
- Great for: New Year goals, birthday reflections
- Optional: annual "Year in Review" reminder
