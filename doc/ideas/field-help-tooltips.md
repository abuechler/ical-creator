# Field Help Tooltips

## Description
Add small info icons next to form fields that show helpful tooltips explaining each field's purpose.

## User Value
- **Learning**: Understand unfamiliar options
- **Self-service**: No need to search for help
- **Onboarding**: Helps new users

## Implementation Details
- Small (i) icon next to field labels
- Hover/click shows tooltip with explanation
- Examples:
  - Timezone: "Events will be created in this timezone"
  - BYDAY: "Repeats on specific day positions in month"
- Mobile: tap to show, tap elsewhere to dismiss
- Keep tooltips concise (1-2 sentences)
