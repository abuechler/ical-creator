# Floating Action Button

## Description
Add a floating action button (FAB) that stays visible while scrolling, providing quick access to Download action.

## User Value
- **Accessibility**: Always-visible primary action
- **Mobile UX**: Thumb-friendly button position
- **Efficiency**: No need to scroll to download

## Implementation Details
- Circular button fixed at bottom-right
- Download icon, teal color matching theme
- Only appears after form is valid
- Smooth fade-in animation
- Expands to show "Download .ics" on hover
- Hide on desktop if buttons visible in viewport
