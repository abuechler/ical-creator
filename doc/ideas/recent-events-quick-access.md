# Recent Events Quick Access

## Description
Show last 3 recently edited events at the top of the page for quick re-access.

## User Value
- **Speed**: Quick access to recent work
- **Workflow**: Continue where you left off
- **Convenience**: No scrolling to saved events

## Implementation Details
- Horizontal card row below header
- Show 3 most recently edited events
- Compact cards: title + date only
- Click to load into form
- "See all" link to Saved Events
- Scrollable on mobile
